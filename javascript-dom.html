The DOM

From what I currently understand, the 'DOM' stands for 'Document Object Model'. The DOM is basically the tree of a web page, from the grandparents, all the way down to the children. This allows you to access specific elements of a web page with java script or browser web dev tools. You might use it to test your website scaling & functionality, or maybe to play around with the source code without consequence.

The difference between accessing data from arrays and objects

Accessing data from an array is good for when you want to store lots of singlular data and access a specific value. An object can also store lots of data but the data itself means more. It is data that can be associated with another pair of data, like a 'car' that is 'blue'.
